CSS Selector Exercise

In this exercise, you will be completing 6 tasks by editing ONLY the styles.css file. Do not make any changes to this HTML file.

  1. Give the main section 15vw of margin on the left and right sides.
  2. Set the background color of the first, third, and fifth cards to #EEEEEE.
  3. Set the background color of the second, fourth, and sixth cards to #bce5d6.
  4. Make the images inside the first, third, and fifth card circular.
  5. Set the font family of all of the text to 'Courier New'.
  6. Set the font weight of the first paragraph and the paragraph inside the footer to bold. The paragraphs inside the cards should NOT be bold.
